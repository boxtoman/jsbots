#JsBots

Trying to remake this with JavaScriptCore

##System Requirements

Mac OSX 10.9 or newer

##Compiling
Get the source

	git clone https://github.com/boxyoman/jsbots.git

Move into the new directory

	cd jsbots
    
Switch branch

	git checkout JavaScriptCore

Compile

	make

##Running
To run

	./jsbots [bot files]

Example

	./jsbots bot1.js bot2.js
